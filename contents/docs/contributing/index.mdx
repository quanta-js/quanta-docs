# Contributing

We welcome contributions to QuantaJS! Here's how you can help:

## Getting Started

<Stepper>
    <StepperItem title="Fork the Repository">
        Fork the repository on [GitHub - QuantaJS](https://github.com/quanta-js/quanta)
        </StepperItem>
    <StepperItem title="Clone the Repository">
        Clone your fork:
        ```bash
        git clone https://github.com/<your-username>/quanta.git
        ```
        </StepperItem>
    <StepperItem title="Install Dependencies">
        Install dependencies:
        ```bash
        npm install
        ```
        </StepperItem>
    <StepperItem title="Start Development Server">
        Start the development server:
        ```bash
        npm run dev
        ```
        </StepperItem>  
    <StepperItem title="Open in Browser">
        Open [http://localhost:3000](http://localhost:3000) in your browser.
        </StepperItem>
</Stepper>

## Submitting Changes

- Open a pull request with a clear description of your changes.
- Follow the Conventional Commits format for commit messages.

## Reporting Issues

- File bugs or suggestions on the [GitHub Issues](https://github.com/quanta-js/quanta/issues/new) page.
